## What This Skill Provides

**7 Commands:**
- `check-prerequisites` → Verify installation
- `install` → Install Prometheus + Node Exporter
- `configure-scrape` → Add scrape targets
- `add-exporter` → Setup exporters
- `query-metrics` → Run PromQL queries
- `test` → 5-test suite
- `troubleshoot` → Auto-fix issues

**Pull-Based Model:**
- Prometheus scrapes /metrics every 15s
- Targets expose metrics HTTP endpoints
- Time-series database storage
- PromQL query language
