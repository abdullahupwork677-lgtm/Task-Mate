# Prometheus Monitoring - Quick Start

**One-command Prometheus - No monitoring specialist needed!**

## 🚀 Quick Usage

### 1. Install Prometheus
```bash
python3 scripts/tool.py install --method docker
```

### 2. Configure Scrape Targets
```bash
python3 scripts/tool.py configure-scrape --target localhost:8080
```

### 3. Test
```bash
python3 scripts/tool.py test
```
