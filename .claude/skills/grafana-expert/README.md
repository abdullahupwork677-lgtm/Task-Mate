# Grafana Expert - Quick Start

**One-command Grafana - No monitoring specialist needed!**

## 🚀 Quick Usage

### 1. Install Grafana
```bash
python3 scripts/tool.py install --method docker
```

### 2. Setup Datasource
```bash
python3 scripts/tool.py setup-datasource --url http://localhost:9090
```

### 3. Test
```bash
python3 scripts/tool.py test
```
